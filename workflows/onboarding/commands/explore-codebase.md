# Explore Codebase

## Steps

1. **Overview**: Read CLAUDE.md and README for project summary
2. **Architecture**: Identify main modules and their relationships
3. **Entry points**: Find the main execution paths
4. **Conventions**: Summarize code style, testing, and git workflow
5. **Key files**: List the 5-10 most important files to understand first
6. **Development setup**: How to install deps, run tests, and start developing

# Sessions

## Steps

1. **List recent sessions**: Show last 10 sessions with timestamps and summaries
2. **Session detail**: For a selected session, show:
   - What was worked on
   - Files modified
   - Commits created
   - Context that was saved
3. **Restore context**: Load a previous session's context into the current session
4. **Export**: Save session summary to `.claude/session-context.md`

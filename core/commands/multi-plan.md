# Multi-Plan

## Steps

1. **Understand scope**: What is the full task? What are the deliverables?
2. **Decompose**: Break into independent subtasks (aim for 3-7 subtasks)
3. **Assign agents**: Map each subtask to the best-fit agent
4. **Identify dependencies**: Which tasks must complete before others can start?
5. **Create execution waves**: Group independent tasks into parallel waves
6. **Define handoffs**: What context passes between dependent tasks?
7. **Output plan**: Ordered task list with agent assignments, dependencies, and wave groupings

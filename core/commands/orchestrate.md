# Orchestrate

Break a complex task into parallel subtasks and dispatch agents.

## Steps

1. **Decompose**: Break the task into independent subtasks
2. **Assign**: Map each subtask to the best agent (reviewer, debugger, architect, etc.)
3. **Dispatch**: Launch agents in parallel for independent tasks, sequentially for dependent ones
4. **Collect**: Gather results from all agents
5. **Synthesize**: Combine results into a coherent response
6. **Verify**: Check for conflicts or inconsistencies between agent outputs
